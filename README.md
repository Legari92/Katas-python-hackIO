# Katas de Python – Fundamentos de Programación

Este repositorio contiene la resolución de una colección de katas de Python realizadas como parte del curso, orientadas a reforzar los fundamentos del lenguaje y el razonamiento lógico aplicado a la programación.

Los ejercicios se centran en el uso correcto de la sintaxis, el control del flujo del programa y la resolución de problemas básicos y progresivos.

---

## Objetivo

Practicar y consolidar conceptos fundamentales de Python, entre ellos:

- Uso de condicionales (if, elif, else)
- Definición y uso de funciones
- Trabajo con listas, tuplas, diccionarios y cadenas de texto
- Uso de funciones como map, filter y reduce
- Manejo de excepciones
- Entrada de datos por teclado
- Lógica de decisión y validación
- Programación orientada a objetos

---

## Contenido

El archivo principal (`katas.ipynb`) incluye la resolución de múltiples katas independientes del curso, que abarcan distintos tipos de ejercicios de lógica y programación en Python.

Cada kata sigue el enunciado proporcionado y se resuelve de forma directa, priorizando la claridad y el cumplimiento de los requisitos.

---

## Requisitos

- Python 3.x
- Jupyter Notebook o Jupyter Lab para abrir el archivo `.ipynb`
- No se utilizan librerías externas fuera de la biblioteca estándar

---

## Ejecución

Abrir el archivo `katas.ipynb` con Jupyter Notebook o Jupyter Lab y ejecutar las celdas.

Algunos ejercicios solicitan datos al usuario mediante `input()`.

---

## Notas

Las soluciones están enfocadas en el aprendizaje y en cumplir los enunciados del curso, sin añadir complejidad adicionales.
